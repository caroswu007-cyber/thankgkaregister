使用说明

  1. 报名总览数据实时来源于 Supabase 数据库；本页不另存任何报名信息，关闭页面无影响。
  2. 需要离线归档或同步到腾讯表格，点「下载全部 CSV」即可，腾讯表格中走 文件 → 导入。
  3. 误提交或测试报名可在总览表格中点「删除」清理；删除前建议先导出 CSV 备份。
  4. 学员自助查询页 query.html 优先读 Supabase RPC，备用 data/students.json。

TSV 行 → 学员 JSON 片段（辅助）

从腾讯表格或邮件中复制一整行数据（Tab 分隔），生成可合并进 students 数组的对象草稿。请核对 currentStatus、courseHistory 等字段后再入库。
