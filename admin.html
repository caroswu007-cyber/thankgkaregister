腾讯文档与学员查询数据

  1. 无国外注册（推荐）：邮件中复制多行 TSV → 下方「导出到腾讯表格」→ 下载 CSV → 在腾讯表格使用文件 → 导入合并到台账。适合管理员偶尔批量处理。
  2. 可选：js/sync-config.js 中的 Supabase 为境外服务；若不需要可留空，仅走邮件 + CSV。
  3. 学员查询仍依赖 data/students.json；可用下方「TSV → JSON」维护后发布站点。

导出到腾讯表格（国内、无国外注册）

将邮件里多行 TSV（每行一条报名）粘贴到下方框内，点下载 CSV。然后在腾讯文档 / 腾讯表格中通过文件 → 导入 → 本地文件（或工具栏中的「导入数据」）选择该 CSV，按向导合并到当前表格。全程使用腾讯官方能力，无需注册国外服务。

  1. 下载的 CSV 已带与站内一致的表头行；若表中已有表头，导入时可选择「跳过重复表头」或先导入到新子表再复制。
  2. 若用桌面 Excel 打开乱码，请优先在腾讯表格网页版直接导入 UTF-8 CSV。

TSV 行 → 学员 JSON 片段（辅助）

从腾讯表格或邮件中复制一整行数据（Tab 分隔），生成可合并进 students 数组的对象草稿。请核对 currentStatus、courseHistory 等字段后再入库。
