## 技术方案（已确定）

| 功能 | 技术 |
|---|---|
| 页面框架 | 纯 HTML + CSS + 原生 JS |
| 自动确认邮件 | EmailJS（免费，无需后端） |
| 管理员数据管理 | 腾讯文档（台账）+ 可选 **Supabase 自动入库**（见 `docs/supabase-autosync.md`）+ 可选 **腾讯智能表服务端推送**（见 `docs/tencent-server-push.md`） |
| 学员查询数据 | data/students.json（从腾讯文档导出） |
| 身份证隐私保护 | SHA-256 哈希匹配，不存原文 |

## 报名表单字段

- 姓名（必填）
- 身份证号（必填，用于学员自助查询）
- 手机号（必填）
- 微信号（必填）
- 年龄（必填，≥16岁）
- 电子邮件（必填，接收确认邮件）
- 性别
- 所在城市
- 是否有绘画基础
- 健康声明勾选（征得家属同意 / 身体健康 / 知晓活动规则）

---

## 学员查询功能

输入：姓名 + 身份证号（后6位）

展示：
- 本期报名状态（已确认 / 候补 / 未报名）
- 历史参课次数
- 历史参课记录（期次 + 日期）
- 联系信息确认（手机号中间隐藏）

## EmailJS 配置（开始编码后需要用户操作）

1. 注册 https://www.emailjs.com 账号（免费）
2. 添加邮件服务（QQ邮箱 / 163邮箱）
3. 创建两个模板：
   - 确认邮件（发给报名者）
   - 管理员通知邮件（发给管理员）
4. 将 Service ID / Template ID / Public Key 填入代码配置区
